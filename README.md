# Моделирование базовых локомоторных паттернов роботизированного агента в симуляционной среде CoppeliaSim
### Описание проекта

Данный репозиторий посвящен выпускной квалификационной работой по направлению 09.04.02 Информационные системы и технологии студента Кипиной А.В.по теме "Моделирование базовых локомоторных паттернов роботизированного агента в симуляционной среде CoppeliaSim". Разработанный и выложенный здесь инструмент предназначен для воспроизведения паттерна ходьбы домашней кошки роботизированным агентом в специализированной среде

### Метрики

| Модель      | MAE (рад) | MaxErr (рад) | R2     |
|-------------|-----------|--------------|--------|
| WalkingNet  | 0.001     | 0.2186       | 0.9978 |
| CPGNet      | 0.001     | 0.2167       | 0.9978 |
| WalkingLSTM | 0.0042    | 0.4695       | 0.9557 |
